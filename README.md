# TODO List x Sequelize

此專案主要透過 mySQL 與 Express 建立用戶資料，並透過 bcrypt 加密及雜湊提高密碼安全性，且使用 passport 套件新增本地輸入跟 FB 輸入。

# 安裝流程

1. git clone 下載檔案

   ```
   git clone https://github.com/Ray7808/todo-sequelize.git
   ```

2. 安裝相關套件

   ```
   npm install
   ```

3. 執行程式
   ```
   npm run start
   ```
